My Projects

Kankoku

Designed to assist users in choosing an Anime series to watch, Kankoku implements the Jikan API to generate data based on the user's search. Our team set out to create a website centered around providing all the information needed when embarking on the search for a new Anime series to enjoy. After entering a specific series into the search bar, the Jikan API generates all available related results and displays the synopsis, ratings, and reviews for each.

Knitlify

There's no better hobby than needle-work, but where do you start? All the questions I asked when I first got into it, I've attempted to answer with my own little creation, Knitlify. With easily accessible links to beginner articles for multiple types of the craft, as well as features intended for more seasoned hobbyists, Knitlify is a one-stop-shop for any interested user.

Sen-Pie

Is it okay to create an entire website based on a pun? I hope so, since that is exactly how Sen-Pie was started. Put a team of anime-lovers and mix some database and Express magic in, and you've got a recipe for a great and interactive ordering site. Sen-Pie includes a login, sign-up, home, checkout, and confirmation page all centered around creating an easy shopping experience.

Forte

Perfect for music-lovers and procrastinators, Forte is a piano-simulator app that allows users to create sounds with their own computer keyboard. Created with React, Forte uses the soundfont-player API to provide all the notes in correspondence with each piano key.
